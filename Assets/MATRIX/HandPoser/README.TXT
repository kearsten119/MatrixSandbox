How To Use

1 - Instantiate a HandPoser object from the prefab.
2 - Run the project.
3 - Modify the pose of the HandPoser object through the HandPoser script front panel controls.
4 - When the hand is positioned as desired, copy the HandPoser object in the hierarch (Ctrl-C).
5 - Stop running the project.
6 - Paste the HandPoser object (Ctrl-V) into the hierarchy.  By default, this object will be in the desired pose.
7 - Disable the Animator in the pasted object, if it isn't already disabled.
8 - Repeat steps 2 - 7 to create additional poses.
9 - Delete or inactivate the HandPoser object from Step 1
